按你的方向——**给自己逛、像空间、多风格、看展各展各的味道**——我按气质分了类。不必全都像，但每类都能借一点。

## 一、最接近「可以逛的空间」

### [Ring Hyacinth / 海辛 · hyacinth.im](https://hyacinth.im/)
可操控小猫在 3D 广场里逛：作品展板、马戏影院、小电台、关于我的碎片散落各处。  
**像什么**：个人博物馆 + 游戏式门厅，内容分区靠「走」发现，不是列表。  
**可借鉴**：wing 用「区域」组织；每个点位一种呈现（展板 / 影院 / 便签）。

### [TheYellowDuck · Portfolio Museum](https://github.com/TheYellowDuck/portfolio-website)（开源，可搜 live demo）
像素风博物馆：WASD 走动，靠近展品按 E 查看；网页版与游戏版同一套内容。  
**像什么**：轻量「知识宫殿」——技术不重，隐喻很直观。  
**可借鉴**：门厅 + 可逛；ghost 访客轨迹（多人不互动也有「有人来过」感）。

### [Balairung · 3D Portfolio Hall](https://github.com/Muhammad-Hazimi-Yusri/portfolio-hall)
滚动驱动的 3D 博物馆导览，也可切第一人称自由探索。  
**像什么**：导览册 + 展厅，默认被带着走，熟客可自己乱逛。  
**可借鉴**：Tour mode / Free-roam——路人友好，你自己可深挖。

### [Arthur Torres · Mountain Portfolio](https://github.com/ArthurTorres75/mountain-portfolio)
3D 小镇，不同区域对应 about / skills / projects；点地标进独立 station 页。  
**像什么**：**空间导航 + 每区不同页面**——和你「wing → room → 不同风格」很像。  
**可借鉴**：地图上一点，换一套完整页面（不是只换内容）。

## 二、最接近「策展 / 看展 / 图文随拍」

### [Craig Mod · craigmod.com](https://craigmod.com/)
作家 + 摄影师，长篇图文像书；徒步日本、喫茶、地方志。  
**像什么**：每次出行是一册**独立出版物**，不是「又一篇 blog post」。  
**可借鉴**：每次看展 = 一本小书；首页像书架，不是时间流。

### [walkkumano.com（Koya Bound 等）](https://walkkumano.com/)
某次徒步的专属站：地图、GPX、照片、日志在同一叙事里。  
**像什么**：**一次展览一个站**——和你 `2026-07-14/玩偶艺术展` 同构。  
**可借鉴**：展间可有独立 slug，不必全塞进统一模板。

### [Hitotoki · hitotoki.org](https://hitotoki.org/)
地点 + 短文字 + 照片 = moment；可慢慢长成 longer narrative。  
**像什么**：碎片先上墙，以后再策展成展。  
**可借鉴**：日常随拍先「挂单」，以后收成正式展间。

## 三、最接近「知识宫殿 + 艺术感」

### [Maggie Appleton · maggieappleton.com](https://maggieappleton.com/)
数字花园 + 手绘插图；笔记互链，视觉统一但每篇有叙事插图。  
**像什么**：图书馆 wing——安静、可生长、有美学系统。  
**可借鉴**：wing 内统一视觉宪法；room 之间用插图 / 配色微调，而非完全换皮。

### [Gwern · gwern.net](https://gwern.net/)
一人百科全书：排版、脚注、暗色、信息密度都极个人。  
**像什么**：档案馆 wing——不讨好读者，但极有身份感。  
**可借鉴**：「主要给自己看」时可以更密、更怪、更完整。

### [Frank Chimero · frankchimero.com](https://frankchimero.com/)
设计师个人站，每篇像独立页面，多年仍好读。  
**像什么**：随笔 wing 的标杆——**文章即作品**。  
**可借鉴**：稿件 / 随笔不必和看展共用 layout。

## 四、最接近「风格鲜明、怪、实验」

### [Neocities · Browse](https://neocities.org/browse)
大量「不像博客」的个人站：像素、闪烁、异形布局、私人互联网感。  
**像什么**：每个站是一个**人格房间**，不追求统一 UX。  
**可借鉴**：某 wing（如看展）可走更野；门厅负责把人接住。

### [Brutalist Websites](https://brutalistwebsites.com/)
策展式目录，收录反精致、反模板个人站。  
**像什么**：找灵感用的「参考美术馆」，不是单站。  
**可借鉴**：当你想某次展更怪诞时，来这里找排版和勇气。

### [sadgrl.online（及同类 Neocities 站）](https://sadgrl.online/)
个人互联网美学：主页像房间，链到 guestbook、图库、manifesto。  
**像什么**：**门厅即人格**——访客一眼知道这是谁的空间。  
**可借鉴**：首页不必是文章列表，可以是「欢迎来到我家」。

## 六、和「不同展位不同风格」最相关的组合

若只选 **5 个**当 mood board：

1. **[hyacinth.im](https://hyacinth.im/)** — 可逛、分区、每区不同媒介  
2. **[craigmod.com](https://craigmod.com/) + [walkkumano.com](https://walkkumano.com/)** — 一次出行 / 一次展 = 独立世界  
3. **[maggieappleton.com](https://maggieappleton.com/)** — 花园 wing、系统美学  
4. **[Arthur Torres mountain portfolio](https://github.com/ArthurTorres75/mountain-portfolio)** — 空间 → 不同 station 页  
5. **[Neocities browse](https://neocities.org/browse)** — 允许怪、允许实验  

你的 **玩偶艺术展页** 已在 Craig Mod「一次展一个站」和 Neocities「强风格 room」之间——缺的是 **门厅** 和 **展间索引**，不是审美方向。

## 一句实话

和你想法**整体最像**的，往往不是某一个传统博客，而是：

- **Craig Mod 系**（每次经历 = 独立策展）  
- **海辛 / 3D 广场系**（门厅可逛、点位异质）  
- **Neocities 系**（风格敢做、主要取悦自己）  

GitHub Pages 完全能承载这个方向；上面这些人多数也是**静态站 + 强策展 + 按项目换视觉**。

若你愿意，下一轮可以只做一件事：从上面挑 **2～3 个你最有感觉的**，我帮你拆成「门厅 / 看展 wing / 展间卡片」的具体界面草图（仍只讨论、不写代码）。
