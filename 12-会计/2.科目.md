# 会计科目分类

|              | 会计科目                              |
| ------------ | ------------------------------------- |
| 资产类       | 会计要素： **资产**                   |
| 负债类       | 会计要素： **负债**                   |
| 所有者权益类 | 会计要素： **所有者权益、利润**       |
| 成本类       | 会计要素： **费用（归属于成本部分）** |
| 损益类       | 会计要素： **收入、费用**             |

# 会计科目（常用整理）

会计科目是对会计要素进一步分类的名称。实务中按 **《企业会计准则》** 常用编号习惯大致为：1 资产、2 负债、3 共同、4 所有者权益、5 成本、6 损益（不同企业明细会略有差异，以本单位科目表为准）。

## 三、共同类（金融企业为主）

一般工商企业较少用；银行、证券、保险等行业会大量使用，如 **清算资金往来、货币兑换、衍生工具、套期工具** 等。初学可先记住「存在这类别」，遇到金融实务再细查。

## 与「会计要素」的对应关系（复习）

| 要素       | 主要落在哪些科目                            |
| ---------- | ------------------------------------------- |
| 资产       | 资产类（含备抵科目如累计折旧、坏账准备）    |
| 负债       | 负债类                                      |
| 所有者权益 | 所有者权益类                                |
| 收入       | 损益类中的收入、利得相关科目                |
| 费用       | 成本类 + 损益类中的成本、费用、损失相关科目 |

---

## 使用说明

1. **编号**：不同软件、行业可能增设二级、三级科目，一级科目名称以财政部应用指南和本单位制度为准。  
2. **备抵科目**：如累计折旧、坏账准备，余额方向与所备抵的资产相反，但在资产负债表上与相关资产 **抵减后** 列示。  
3. **本年利润**：期末将收入、费用转入后，再转入 **利润分配**，年度中间不直接把损益类余额结到「利润分配」时，往往先经「本年利润」。

如需把某几个科目（例如你们公司账套里实际启用的）单独摘成一页「工作用速查表」，可以列出名称，我可以按你的格式再压一版。
