# Good Morning ☀️

## 本地预览

先生成首页和目录数据：

```bash
node .github/workflows/index.mjs
```

再启动静态服务器：

```bash
python3 -m http.server 8080
```

打开 http://localhost:8080 预览。

不要直接双击打开 `index.html`，页面需要通过 HTTP 读取 `structure.json` 和文章文件。
